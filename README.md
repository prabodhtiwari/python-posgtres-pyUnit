## Python-Postgres-PyUnit

Basic Python project that connects with postgres and tests output.

Requirments:
    > `Python 2.7`
    > `Postgres`

Environment Variables:

    > `DB`
    > `HOST`
    > `PASSWORD`
    > `USER`
    > `PORT`


To install dependencies:
    > `pip install -r requirements.txt`

To run:
    > `python test.py`
